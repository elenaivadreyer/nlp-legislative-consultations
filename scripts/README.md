# Scripts

This directory contains Python scripts for the NLP pipeline:

- **data_acquisition**: Scripts for loading data from Excel files and web sources
- **text_preprocessing**: Scripts for text cleaning, tokenization, and normalization
- **modeling**: Scripts for text understanding, feature extraction, and model training
- **evaluation**: Scripts for model evaluation and metrics

## Usage

Run scripts from the project root directory:

```bash
python scripts/your_script.py
```
